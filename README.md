# Iternum Agency Cockpit

Live custom department dashboards for Iternum Digital.

## Current version
This repo contains a static frontend that fetches live data directly from the public Google Sheets sources for:
- Google Ads
- SEO
- Social Media

## Views
- COO Master
- Google Ads
- SEO
- Social Media

## Notes
- This version is frontend-only and designed for simple static hosting.
- It uses live Google Sheets data via the Google Visualization endpoint.
- Main file: `index.html`
